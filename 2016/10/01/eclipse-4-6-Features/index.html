eclipse 4.6 Features

2016-10-01 0 Comments Word Count: 691(words) Read Count: 2(minutes)

由于最近工作需要，将eclipse内核升级，所以写一篇关于eclipse4.6一些新特性（基本翻译了一下官网的相关doc）

一、Platform and Equinox

  • Eclipse4.6的组件SWT现在在Windows和Linux对于high-DPI显示器上的图像会做一个平衡处理,类似于Mac OS x上的视网膜支持。在缺乏高分辨率图像,SWT会自动扩展可用的图像,以确保基于SWT应用程序像Eclipse这样的缩放比例的分辨率显示器。

    下图是4.5上的效果，可以做一个对比

  • 添加自动换行按钮，快捷键（Alt+Shift+Y）

  • 切换编辑器字体大小快捷键（Ctrl=放大）（Ctrl-缩小）

  • 自动保存代码设置（Preferences > General > Editors > Autosave）

    默认是不保存，你可以设置为自动保存，并添加自动保存的时间

  • 通过选项搜索二进制文件

  • Terminate and Relaunch from history
    A Terminate and Relaunch option is now available while launching from history. The default setting is to launch without terminating previous launches. To enable automatic termination, select the option “Terminate and Relaunch while launching from history” on Preferences > Run/Debug > Launching.

  • 选择从历史的工作空间中启动（可以删除）

  • 在窗口左上角开头显示工作空间的名字

    新版本可以通过 Preferences > General > Workspace 设置窗口开头的名字，当然也有其他设置

  • 快速查询（Ctrl + 3）

    这项功能对于eclipse高手来说是非常有用的。例如下面这项功能

  • 隐藏工具栏

    高手通过Ctrl+3 ，在输入 Toggle visibility of the window toolbars，就可以触发隐藏工具栏的按钮。

  • 隐藏视图的名字

    新版为了追求简洁，默认把perspective的名字去除，既可以显示多个视图，也显美观简洁

  • 全屏
    win：Alt+F11 (Window > Appearance > Toggle Full Screen)
    mac：Control+Command+F (Window > Toggle Full Screen )

  • 添加是否选择主题设置的选项

  • 文件类型关联设置

    通过Preferences > General > Editors > File Association设置类型

    当选中文件，右键Open With > Other... 可以以你设置好的类型打开

  • 代码模板（template）

    对date等变量添加可设置的参数

  • import project

    导入工程可以是某个文件，也可以是某个zip压缩包

  • new选项中新建项目加入指定working set

  • eclipse主题：dark

    这次新出的dark主题，有点模仿IDEA，但是个人感觉有点烂，毕竟有很多显示还是没做全，不属于eclipse的风格。只不过有一点要说，dark主题下的侧边的滚动条是有特点的，你也可以通过-Dswt.enable.themedScrollBar=false在eclipse.ini设置去除

  • 其他
    eclipse welcome
    Improved menu support on GTK3
    Tweaking SWT’s auto-scaling
    Advanced capabilities preferences are now displayed in place (no extra dialog)
    Java 9 previews
    Ant 1.9.6 Eclipse has adopted Ant version 1.9.6.
